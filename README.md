# Repositorio testigo de mi aprendizaje en rust

En este repositorio subiré tanto los scripts de las lecciones como los ejercicios propuestos en los *materiales de aprendizaje*.

## Materiales de aprendizaje

Por el momento, mi ruta contempla:
**Comprhensive Rust by Google**
**The Rust Programming Language** (el libro virtual de *Steve Klabnik, Carol Nichols, Chris Krycho y la comunidad de Rust*)

## Estructura del proyecto

```text
Aprendizaje_Rust/
├── comprehensiveRust_byGoogle/    # Notas del curso de Google
│   ├── clases/                    # Scripts sueltos (.rs) de teoría
│   └── ejercicios/                # [Cargo Project] Retos del curso
├── Rust_lang_book/                # Seguimiento del "Libro de Rust"
│   ├── clases/                    # Conceptos de sintaxis y memoria
│   └── ejercicios/                # [Cargo Project] Proyectos del libro
├── .gitignore
└── README.md
```
